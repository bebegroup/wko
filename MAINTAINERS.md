# Maintainers

| GitHub | Vai trò | Khu vực |
|---|---|---|
| [@saucevn](https://github.com/saucevn) | Lead maintainer, project owner | Toàn bộ repo |

## Trở thành co-maintainer

Cần:

- ≥ 5 PR merged không trivial (không chỉ typo/comment)
- ≥ 1 năm active trong project (commits, issue triage, PR review)
- Lead maintainer mời

## Liên lạc

- 🐛 Bug/feature: [GitHub Issues](https://github.com/saucevn/wko/issues)
- 🔒 Security: security@saucevn.dev
- 💬 General: [GitHub Discussions](https://github.com/saucevn/wko/discussions)

## Decision-making

- **Bug fix / docs**: Maintainer tự merge sau review
- **New feature** (backward compatible): 1 maintainer approve
- **Breaking change**: RFC discussion 7+ ngày, lead maintainer quyết định
- **Security**: Lead maintainer quyết định, không qua public PR (private fix → CVE → release)
